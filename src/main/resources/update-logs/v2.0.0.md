# 星光小栈 V2.0.0 正式上线

🎉 经过几个月的开发，星光小栈 V2.0 正式上线了！

## 重大变更

### 全新架构

- 后端升级到 Spring Boot 3.2.5
- 前端使用 Vue 3 + Vite + TypeScript 重构
- 数据库升级到 MySQL 8.0
- 引入 Redis 缓存

### 技术栈更新

**后端技术栈：**
- Spring Boot 3.2.5
- Spring Security + JWT
- Spring Data JPA
- MySQL 8.0 + Redis
- Flyway 数据库迁移
- MapStruct 对象映射
- WebSocket 实时通知

**前端技术栈：**
- Vue 3.4 Composition API
- Vite 5 构建工具
- TypeScript 类型安全
- Vue Router 4 路由管理
- Pinia 状态管理
- Axios HTTP请求
- markdown-it Markdown渲染
- highlight.js 代码高亮

## 部署方式

### Docker支持

提供完整的Docker镜像和docker-compose配置：

```bash
docker-compose up -d
```

### 传统部署

支持传统的JAR包部署方式：

```bash
java -jar blog-backend.jar
```

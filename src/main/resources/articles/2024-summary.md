# 我的2024年度总结

2024年即将结束，回顾这一年，有收获也有遗憾，有成长也有不足。

## 技术成长

### 学习与实践

今年重点学习了以下技术：

1. **Spring Boot 3.x**
   - 完成了从2.x到3.x的迁移
   - 深入理解了原生镜像和可观测性
   - 在生产环境中成功应用

2. **Vue 3 Composition API**
   - 重构了几个老项目
   - 编写了多个可复用的Composables
   - 体会到了Composition API的优势

3. **Docker & Kubernetes**
   - 实现了应用的容器化部署
   - 学习了K8s的基本概念和操作
   - 搭建了完整的CI/CD流程

4. **Redis深入应用**
   - 解决了几个缓存相关的问题
   - 实现了分布式锁
   - 优化了系统性能

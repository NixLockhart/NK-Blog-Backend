# 欢迎来到星光小栈

欢迎来到星光小栈V2.0！这是一个全新架构的个人技术博客，旨在分享技术知识和生活感悟。

## 博客特色

### 技术栈

星光小栈V2.0采用了现代化的技术栈：

- **后端**: Spring Boot 3.2.5 + MySQL 8 + Redis
- **前端**: Vue 3 + Vite + TypeScript
- **部署**: Docker + Nginx

### 核心功能

1. **文章系统**
   - 支持Markdown编写
   - 分类管理
   - 全文搜索
   - 代码高亮

2. **评论系统**
   - 支持楼中楼回复
   - Markdown格式
   - 访客信息记录

3. **留言板**
   - 访客留言
   - 友情链接展示

4. **统计分析**
   - 访问量统计
   - 文章热度排行
   - 数据可视化

## 如何使用

### 浏览文章

点击首页的文章卡片即可查看文章详情。支持按分类筛选、搜索文章。

### 发表评论

在文章详情页底部可以发表评论，支持Markdown格式。

### 留言互动

访问留言板页面，可以给博主留言或申请友情链接。
